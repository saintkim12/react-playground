# 02-try-todolist
공식 문서의 [주요 개념](https://ko.reactjs.org/docs/hello-world.html)을 읽고 만든 TodoList App
Create React App으로 생성하였음

## 구성

### components
각 부분을 컴포넌트화함 

#### TodolistApp.js
전체 앱 컴포넌트

#### SummaryForm.js
요약 정보 컴포넌트

#### AddItemForm.js
항목 추가 폼 컴포넌트

#### SearchForm.js
검색 폼 컴포넌트

#### ListForm.js
데이터 리스트 출력 폼 컴포넌트

#### ItemWrap.js
리스트의 항목을 그룹화해 감싸는 컴포넌트

#### ItemRow.js
각 항목에 대한 컴포넌트
